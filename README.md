# Red Bull Racing RB20 - Scrollytelling Experience

A high-performance, premium web experience dedicated to the **Red Bull Racing RB20** Formula 1 car. Built with cutting-edge web technologies, this project offers an immersive "scrollytelling" journey through the car's specifications, performance metrics, and driver lineup featuring Max Verstappen and Isack Hadjar.

## 🌟 Features

- **Immersive Hero Section**: A stunning introduction with dynamic typography and high-resolution imagery.
- **Apple-Style Scroll Reveal**: Smooth, engaging animations that reveal the RB20's design as you scroll.
- **Performance Data**: An interactive grid showcasing the staggering statistics and performance metrics of the car.
- **Technical Specifications**: Detailed insights into the engineering marvels under the hood of the RB20.
- **Driver Lineup**: Dedicated profiles for Max Verstappen and Isack Hadjar, complete with biographical details and stunning photography.
- **Legacy Section**: A tribute to the ongoing dominance and history of Red Bull Racing.

## 🚀 Tech Stack

This project was built with a focus on blazing-fast performance, smooth animations, and a modern developer experience:

- **Framework**: [Next.js 16](https://nextjs.org/) (App Directory)
- **Library**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Language**: TypeScript

## 🏎️ Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Ensure you have Node.js (v20 or newer recommended) installed on your local machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/redbull-racing.git
   ```

2. Navigate to the project directory:
   ```bash
   cd redbull-racing
   ```

3. Install the dependencies:
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   ```

5. Open your browser and navigate to `http://localhost:3000` to see the site in action!

## 📂 Project Structure

- `app/`: Next.js app directory containing the main `page.tsx`.
- `components/`: Reusable Framer Motion animated React components (`Navbar`, `HeroSection`, `DriverCard`, `StatsGrid`, etc.).
- `data/`: Structured context data mapping for the RB20 specs.
- `public/`: Static assets like driver images (`max.png`, `isack.png`) and general project references.
